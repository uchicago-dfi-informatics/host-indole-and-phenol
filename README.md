# Digestion-resistant proteins support the healthy metabolite profiles associated with plant-based diets

This repository contains metagenome and metabolome data used in this manuscript.

The shotgun metagenome and metabolome data was collected from the patients who underwent liver transplant. One sample per patient nearest to the liver transpant event was selected. The samples were further divided into two cohorts: 1) Samples that contained Oscillospiraceae family of bacteria and the samples that didn't contain Oscillospiraceae family. 

The <B>data</B> folder contains two sub-folders:<BR><BR>
<B>1. metagenome: </B> It contains two files, Metphlan4 results for all the shotgun metagenome samples that contained "Oscillospiraceae" and Metaphlan4 results for all samples with no "Oscillospiraceae" present.<BR><BR>

<B>2. metabolome: </B> It contains the metabolites in DFI's PFBBr panel (SCFAs) and their concentration values for the two cohorts selected (with and without Oscillospiraceae).
